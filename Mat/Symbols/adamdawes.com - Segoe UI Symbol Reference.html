Segoe UI Symbol Reference

One of the things I quickly realised when I started creating Windows 8 applications is that there is a lot of content hidden away in the Segoe UI Symbol font. The characters from this font can be found scattered throughout the Windows 8 user interface, and most of the buttons that can be found in the application bar use images supplied by this font.

Clearly in order to be able to use the font effectively, it is important to know which symbols are contained within it. Some time ago I found an excellent page on Richard Garside's web site, www.nogginbox.co.uk, entitled Creating app bar buttons in WinRT. Within this page is a table containing a complete list of all Segoe UI Symbol characters. I used this a lot and found it very useful, but one thing it seemed to be missing was details about the names of the buttons within the Common/StandardStyles.xaml file. Without these names, it was hard to search through the symbols looking for one that might suit my particular requirement.

I therefore decided to create an enhanced version of this table, which includes all of the symbol names, where available. The results of this are below. Full credit to Richard for providing the original idea and the list of "un-named" symbols, which I have included below. Perhaps one day I'll work through these and add names to them to make them searchable too.

This page will only display properly if you have the Segoe UI Symbol font installed on your computer. This should be installed by default if you are using Windows 7 or Windows 8.
